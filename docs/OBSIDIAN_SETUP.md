# 옵시디언 Vault 설정

## 설정 방법
1. 옵시디언 열기
2. "Vault 열기" → 기존 폴더 선택
3. C:/Ai_Wiki/viralboard/docs 선택
4. 완료 — git commit 할 때마다 옵시디언에 자동 반영

## 폴더 구조
docs/
├── PROJECT_STATUS_YYYY-MM-DD.md  # 프로젝트 현황
├── AI_STUDIO_SPEC.md              # AI Studio 기획서
├── BUSINESS_MODEL.md              # BM 문서
├── DAILY-YYYY-MM-DD.md           # 매일 세션 일지 (자동 생성)
