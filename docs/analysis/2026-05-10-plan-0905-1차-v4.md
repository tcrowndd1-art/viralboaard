```python
import json
from pathlib import Path

CACHE_PATH = Path(__file__).parent.parent / "data" / "invalid_combos.json"
CACHE_TTL_DAYS = 7

def load_invalid_combos():
    if not CACHE_PATH.exists():
        return set()
    try:
        data = json.loads(CACHE_PATH.read_text(encoding='utf-8'))
        cutoff = (datetime.now(timezone.utc) - timedelta(days=CACHE_TTL_DAYS)).isoformat()
        return {(r, c) for r, c, ts in data if ts > cutoff}
    except Exception:
        return set()

def save_invalid_combos(combos):
    CACHE_PATH.parent.mkdir(parents=True, exist_ok=True)
    now = datetime.now(timezone.utc).isoformat()
    data = [[r, c, now] for r, c in combos]
    CACHE_PATH.write_text(json.dumps(data, ensure_ascii=False, indent=2), encoding='utf-8')

INVALID_COMBOS = load_invalid_combos()
```

### G3+ (NEW v4.2) — 정렬 메트릭 = views_per_hour DESC
모든 홈 섹션의 정렬 기준을 단순 `views DESC` → **`views_per_hour DESC` (시간 가중치)**.

```tsx
// 클라이언트 측 계산 (fetch 후)
function viewsPerHour(v: Video): number {
  const hours = Math.max((Date.now() - new Date(v.published_at).getTime()) / 3600000, 1);
  return v.views / hours;
}
const sorted = [...videos].sort((a, b) => viewsPerHour(b) - viewsPerHour(a));
```

이유: views DESC = 5년 전 누적 1억 조회 영상이 어제 100만 영상보다 위. 신선도 회복 안 됨. views_per_hour는 시간 가중 → 신선한 폭발 영상 우선.

### G3++ (NEW v4.2) — 채널 다양화 (DISTINCT ON channel_id)
30일/90일 필터 적용 후 같은 채널 영상이 N개 차지하는 것 방지. 클라이언트에서 1채널 = 1영상 (top viewsPerHour).

```tsx
function uniqByChannel(videos: Video[]): Video[] {
  const seen = new Set<string>();
  return videos.filter(v => {
    if (seen.has(v.channel_id)) return false;
    seen.add(v.channel_id);
    return true;
  });
}
// 정렬 후 unique 적용:
const final = uniqByChannel(sorted).slice(0, perPage);
```

적용 위치: shorts/longform/trending/누적조회수/떡상탐지 모두.
